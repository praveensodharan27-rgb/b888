### Gradients Used:

#### Hero Section:
```css
bg-gradient-to-br from-blue-600 via-blue-700 to-indigo-800
```

#### Trust Badge:
```css
bg-white/10 backdrop-blur-sm border border-white/20
```

#### How It Works Icons:
- Step 1: `from-blue-500 to-blue-600`
- Step 2: `from-emerald-500 to-emerald-600`
- Step 3: `from-purple-500 to-purple-600`

#### Why Choose Us Cards:
- Verified: `bg-blue-50` → `bg-blue-100` (hover)
- Top Rated: `bg-emerald-50` → `bg-emerald-100` (hover)
- Local: `bg-purple-50` → `bg-purple-100` (hover)
- Prices: `bg-amber-50` → `bg-amber-100` (hover)

### Card Pattern:
```tsx
<div className="bg-white rounded-2xl p-6 border-2 border-gray-100 
                hover:border-blue-200 hover:shadow-xl 
                transition-all duration-300">
  {/* Content */}
</div>
```

### Button Pattern:
```tsx
<button className="px-6 py-3 rounded-full bg-blue-600 text-white 
                   font-bold hover:bg-blue-700 transition-all 
                   shadow-md hover:shadow-lg">
  {/* Text */}
</button>
```

### Badge Pattern:
```tsx
<div className="inline-flex items-center gap-2 px-4 py-2 
                rounded-full bg-white/10 backdrop-blur-sm

  {/* Content */}
</div>
```


## 🎯 User Flow Indicators

### Visual Cues:
1. **Numbered Steps** - (1), (2), (3) badges
2. **Arrows** - → for forward actions
3. **Plus Signs** - + for expansion/addition
4. **Check Marks** - ✅ for completed/verified
5. **Stars** - ⭐ for ratings
6. **Emojis** - 🔍 🛡️ 💯 for quick recognition

### Color Coding:
- **Blue** - Primary actions, trust
- **Emerald** - Success, verification
- **Purple** - Premium, special
- **Amber** - Highlights, warnings
- **Red** - Alerts (not used on this page)
